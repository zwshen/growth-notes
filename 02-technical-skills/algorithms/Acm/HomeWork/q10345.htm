Q10345: Crickey/Football Goes Down

孟加拉的板球隊和足球隊在國際比賽中失敗是時常發生的事，人們不知道還可以對他們所愛的隊伍有什麼期待。但是孟加拉的選手在國際程式設計比賽中則有不壞的表現，所以一些人想要將程式設計比賽推廣成像板球或足球那樣普遍。現在他們已經說服了政府官員在球場舉行程式設計比賽。

程式設計比賽按年齡區分可以有6個等級，每個等級所需要的房間都是正方形的，且大小都一樣。球場的形狀則是正圓形。以下的圖說明了如何在球場中蓋1到6個房間，供不同等級的選手競賽。你可以假設圖中的房間看起來對稱，那就真的是對稱。在第6個圖中左下角的房間則是旋轉45度。

給你每個房間的一邊長A，你的任務是算出球場的半徑最少要多少才可以容納的下N（1 <= N <= 6）個房間。

Input

輸入的第一列有一個整數代表以下有多少組測試資料。每組測試資料一列含有1個浮點數A，代表房間的一邊長。

Output

對每組測試資料輸出一列，含有6個浮點數 r1,r2,r3,r4,r5,r6。r1代表建一個房間，r2代表建2間房間，依此類推。請輸出到小數點後11位。請不必擔心浮點數精準度的問題，本題有特殊的程式去判斷。

Sample input

2
0.00000001
0.00000002

Sample Output

0.00000000707 0.00000001118 0.00000001288 0.00000001414 0.00000001581 0.00000001689
0.00000001414 0.00000002236 0.00000002577 0.00000002828 0.00000003162 0.00000003377
