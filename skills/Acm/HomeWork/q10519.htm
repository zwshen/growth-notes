Q10519: !!REALLY STRANGE!!

Raju剛獲得大學理學士的學位。他出社會的第一個工作是到某地區調查該地被劃分為多少塊不同的區域。他發現那裡的人用圓來畫區域，每2個圓一定相交於2點且不會有三個圓相交於同一點。由於人數實在太多了，Raju需要你的幫忙計算出這些圓把這地區切割成多少塊不同的區域。以下的圖為2個圓的情況，共有4個不同的區域。

Input

每組測試資料一列。含有1個整數 n（0 =< n <= 10100）。

Output

對每組測試資料輸出一列，這 n 個圓把這地區切割成多少塊不同的區域。

Sample Input Sample Output

2
3
4
0
12345678987654321

4
8
14
1
152415789666209407864654802316722
