Q583: Prime Factors

因數分解是個常遇到，但是卻不是那麼容易的問題，特別是大數字的時候。
某一個數字n，可以寫成因數分解式：

n = f1 x f2 x … x fk

其中對所有的i來說fi>1,且對所有的i<j來說fi<=fj

Input

輸入包含好幾個資料，每個資料一行，含有1個整數n。（-231<n<231）
n=0代表輸入結束。

Output

對每個測試資料輸出n的因數分解式。請參考sample output

Sample Input

-190 
-191 
-192 
-193 
-194 
195 
196 
197 
198 
199 
200 
0

Sample Output

-190 = -1 x 2 x 5 x 19
-191 = -1 x 191
-192 = -1 x 2 x 2 x 2 x 2 x 2 x 2 x 3
-193 = -1 x 193
-194 = -1 x 2 x 97
195 = 3 x 5 x 13
196 = 2 x 2 x 7 x 7
197 = 197
198 = 2 x 3 x 3 x 11
199 = 199
200 = 2 x 2 x 2 x 5 x 5
