Q10361: Automatic Poetry

"喔！又是這種謎語！"羅拉說。

在電腦遊戲"古墓奇兵14"中，羅拉如同往常在埃及古墓中奔跑。現在她站在一道看起來像日耳曼風格的門前，要解決一道謎語才可以過關。這一次的謎語是一種叫做Schuttelreim的詩詞。例子如下：

Ein Kind halt seinen Schnabel nur,
wenn es hangt an der Nabelschnur.    

這種詩詞似乎是典型的德國風格。有趣的是這種奇怪的詩詞如果有人給你第一列，以及第二列的開頭，你就可以自己完成這首詩詞。你的任務就是寫一個程式來自動的完成這首詩詞，所以古墓奇兵中的羅拉可以專心在動作部分，而不用在智力方面傷腦筋。

Input

輸入的第一列有一個整數n，代表以下有幾組測試資料。每組測試資料2列，第一列的格式如下：

s1<s2>s3<s4>s5

其中si可能是空的或小寫的字串或空白。第二列將是小寫的字串或空白而且以...作為結束。每列的長度不會超過100個字元。請參考Sample Input。

Output

對每一組測試資料的2列：l1及l2，你必須按以下的規則輸出2列：c1及c2。

  • 把 l1的"<"和">"拿掉之後，就得到c1。

  • c2 和 l2一樣，但是把 l2 最後的...換成 l1 的 s4s3s2s5。

請參考Sample Output。

Sample input

3
ein kind haelt seinen <schn>abel <n>ur
wenn es haengt an der ...
weil wir zu spaet zur <>oma <k>amen
verpassten wir das ...
<d>u <b>ist
...

Sample Output

ein kind haelt seinen schnabel nur
wenn es haengt an der nabel schnur
weil wir zu spaet zur oma kamen
verpassten wir das koma amen
du bist
bu dist
