Q10391: Compound Words

給你許多字典中的字，你的任務是找出其中所有的2字複合字（two-word compound words）。2字複合字指的是由2個字所合成的字。例如：catfish是由cat及fish這2個字所合成的。注意：這2個字也可以是同一個字。

Input

給你字典的內容，每個字一列，都是小寫字母。且字典中的字已按照字母順序由小到大排好了。字的數目最多不會超過120000個。

Output

輸出字典中所有的2字複合字，按照字母順序由小到大。

Sample Input

a
aa
aaa
aaaa
alien
born
less
lien
never
nevertheless
new
newborn
the
zebra

Sample Output

aa
aaa
aaaa
alien
newborn
