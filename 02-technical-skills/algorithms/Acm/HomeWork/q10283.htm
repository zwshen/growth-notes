Q10283: The Kissing Circles

以下有一些有趣的圖形：你可以發現我們在大圓中分別放入1,2,3,4,5,6......個相同的小圓（灰色部分），每個小圓都和大圓相接觸，且小圓也彼此相接觸。我們的問題是：給你大圓的半徑R以及小圓的個數N，請你求出小圓的半徑r，小圓所為起來部分的面積I（藍色部分），以及在小圓之外，大圓之內的面積E（綠色部分）。

N=1,2,3,4,5,6的圖形

Input

輸入的每一列測試資料有2個非負整數R（ 0 <= R <= 10000）及N（ 1<= N <= 100），分別代表大圓的半徑及小圓的數目。

Output

對每一列測試資料，請輸出r,I,E。到小數點以下10位。

Sample input

10 3
10 4
10 5
10 6  

Sample Output

4.6410161514 3.4732652470 107.6854162259
4.1421356237 14.7279416563 83.8264899217
3.7019190816 29.7315551092 69.1625632742
3.3333333333 45.6568837582 59.0628713615

Attention

使用PI=3.14159可能不夠精確，請用2.0*acos(0.0)作為PI的值。
