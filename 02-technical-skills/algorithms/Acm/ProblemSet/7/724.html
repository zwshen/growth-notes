Reverse 

The Problem 

Write a program which has no input (i.e., contains no input statement - read, readln, scanf, fget, ...) and displays a copy of its source program code in order immediately followed by its source code in reverse. The resulting output will be a palindrome, except that carriage-return line-feed sequences are not reversed.

Abstract example (*** Hypothetical Input ***)  

#include <stdio.h>

int used[10] = {0};
long denominator = 0;

/* Here's a Strange one */
main()
{
   bla;
   bla;
   bla;

   exit (0);
}
