Q10573: Geometry Paradox

在下面的圖中2個小圓彼此接觸，而大圓則又與2個小圓接觸。2個小圓的半徑分別為 r1 和 r2，而2小圓間的共同切線於大圓中的長度為 t。你的任務是算出灰色區域的面積。

Input

輸入的第一列有一個正整數N（N <= 100），代表以下有多少組測試資料。

每組測試資料一列。如果只含有一個整數，那他代表 t，如果含有2個整數，那他代表 r1 和 r2。所有的整數都小於100。

Output

對每組測試資料輸出一列，如果給的資訊足夠回答灰色區域的面積則請輸出該面積（到小數點後4位）。否則請輸出"Impossible"。假設 pi = 2*cos-1(0)。

Sample Input Sample Output 
3
10 10
99
15 20
 
628.3185
3848.8437
1884.9556
