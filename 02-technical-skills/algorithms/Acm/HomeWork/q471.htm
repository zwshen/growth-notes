Q471: Magic Numbers

寫一支程式找出並輸出每一對整數S1及S2符合以下條件：

  1. 無論S1或S2都不可以有任何一位數字重複出現﹔
  2. S1 / S2  = N, N為輸入的數目。

Input and Output

數入的第1列有一個整數代表以下有幾組測試資料。每組測試資料為一個整數N。

輸出為零或更多行，每一行包含S1 / S2  = N，S1 、 S2  及N皆為如上所述的整數。如有兩個以上的解，請由小到大排序。不同的測試資料間請空一列。

3

1234567890

9876543

1234567
