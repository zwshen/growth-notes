Q10215: The Largest/Smallest Box

在下方你會看到一張矩形的的紙張。他的寬度是W，長度是L，而厚度是零。現在要在紙張四個角落沿著點虛線各剪下x*x的正方形。這樣就可以沿著長虛線把這紙張折成一個盒子（沒有蓋子）。

給你L及W，請你算出x的值為多少時，盒子的容量最大及最小？

Input

每筆測試資料一列。每列有2個正的浮點數L,W（0 < L,W < 10000），代表紙張的長度及寬度。

Output

對每一測試資料，請輸一列。每列有3個浮點數（輸出到小數點後3位）。第一個浮點數為x的值為多少時，盒子的容量會最大。第二個浮點及第三個浮點數為x的值為多少時，盒子的容量最小（也就是說盒子容量要最小，x會有2個答案，其中第二個浮點數小於第三個浮點數）。

Sample Input

1 1
2 2
3 3

Sample Output

0.167 0.000 0.500
0.333 0.000 1.000
0.500 0.000 1.500
