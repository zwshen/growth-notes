Q10287: Gifts in a Hexagonal Box

當你要買禮物送給某人時，你也會希望裝禮物的盒子也能漂亮。所以禮物盒子是一件重要的事。現在讓我們回到主題：看以下的圖，分別表示出放置1、2、3、4個同樣大小圓形的禮物在1個正6角行的盒子中。

給你正6角型盒子的一邊長，請你分別求出上圖中各圓形的半徑是多少。

Input

輸入包含多組測試資料，每組測試資料1列。每列有1個不為負的浮點數S（S <= 10000），代表正6角形盒子的一邊長。

Output

對每組測試資料請輸出1列，包含4個浮點數r1,r2,r3,r4。分別代表上圖中正6角形盒子中所放置的 1、2、3、4 個圓形禮物的半徑。請輸出到小數點後10位。

Sample Input

0.0000001
0.0000002
0.0000003

Sample Output

0.0000000866 0.0000000464 0.0000000433 0.0000000375
0.0000001732 0.0000000928 0.0000000866 0.0000000750
0.0000002598 0.0000001392 0.0000001299 0.0000001125
