Q10297: Beavergnaw

當一隻海狸在啃一棵樹的時候，總是對樹幹啃出一個特別的形狀。樹幹剩下的部分看起來像是以一個圓柱所連起的2個圓錐體，並且此圓柱的直徑與高度相同。一隻好奇的海狸想要在樹幹的某特定區域啃掉一固定單位的木材，他要請你幫他計算連接2段樹幹的中央圓柱的直徑應該是多少？

讓我們假設樹幹的直徑為D，一隻海狸在此樹幹啃出來的區段高度亦為D。請問在中央的圓柱部分直徑d應該為多少，才能使這隻海狸啃掉V個單位的木材？請參考下圖：

   

Input

每筆測試資料一列。每列有2個整數D及V。D是長度單位，而V是體積單位。V不會超過海狸所能啃的木材的最大體積。D=0, V=0代表輸入結束。

Output

對每筆測試資料，請輸出圓柱的直徑d，到小數點後3位。

Sample Input

10 250
20 2500
25 7000
50 50000
0 0

Sample Output

8.054
14.775
13.115
30.901
