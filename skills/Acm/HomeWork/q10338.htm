Q10338: Mischievous Children

給一個英文單字，字元經過重新排列後共有多少種組合？

例如：BEE共有3種組合：BEE EBE EEB

Input

第一列有一個正整數代表接下來有幾列測試資料。

測試資料每列有一個單字，最多20個字元，全部為大寫字元（A∼Z）

Output

對每一測試資料，請輸出共有多少種組合。所有的答案都可以以 unsigned long int 儲存，也就是說，我們保證答案一定小於232。輸出格式請參考Sample Output。

Sample Input

4
HAPPY
WEDDING
ADAM
ABC

Sample Output

Data set 1: 60
Data set 2: 2520
Data set 3: 12
Data set 4: 6
