Q10183: How many Fibs?

Fibonacci數列的定義如下：

f1 = 1
f2 = 2
fn = fn-1 + fn-2     ( for n>=3)

給你2個數a和b，請算出有多少個Fibonacci數介於a和b之間。

Input

每一列測試資料有2個整數分別代表a,b（a <= b <= 10100）

a=b=0代表輸入結束。

Output

每列測試資料輸出一列整數，就是共有多少個Fibonacci數（fi）介於a,b之間。（a<= fi <=b）

Sample Input

10 100
1234567890 9876543210
0 0

Sample Output

5
4
