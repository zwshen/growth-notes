Q474: Head / Tails Probability

投擲一枚正常的硬幣n次，出現n次人頭的機率是2-n。給你正整數 n（1 <= n <= 1000000），請你算出2-n。

Input

每筆測試資料一列。每列有一個正整數 n。

Output

對每一列輸入，請輸出2-n，並且以下列的格式來表示：

2^-n = z.xxxe-y

在這裡 z 是一個1∼9的數字，每一個 x 為一個0∼9的數字，y是一個整數。

請參考Sample Output。

Sample Input

1
100
10000
1000000

Sample Output

2^-1 = 5.000e-1
2^-100 = 7.889e-31
2^-10000 = 5.012e-3011
2^-1000000 = 1.010e-301030
