Q10386: Circles in Triangle

在下面的2個圖中分別顯示在正三角形中放置8個及11個相同大小的圓的情形。

給你圓的半徑，請你求出這2個正三角形的邊長。

Input

數入的第一列有一個整數N，代表以下有幾筆測試資料。接下來每筆測試資料一列，有1個浮點數 r（r < 10000），代表放置於三角形中的圓的半徑。

Output

對每一筆測試資料，請輸出2個浮點數，分別代表可以包含8個圓及11個圓的正三角形的最小邊長。請輸出到小數點後10位。

Sample Input

3
0.000000001
0.000000002
0.000000003

Sample Output

0.0000000093 0.0000000107
0.0000000186 0.0000000215
0.0000000279 0.0000000322
