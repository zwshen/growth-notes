Q10450: World Cup Noise

「KO-RE-A, KO-RE-A 」54000名足球迷大聲的叫著，在世界盃足球賽半准決賽中，南韓的球迷聲嘶力竭的為自己國家的球隊加油。雖然他們很興奮，由於其民族性的關係，他們仍然很有組織。例如，他們組織了啦啦隊，用氣笛及呼吼來加油。

那些氣笛聲是由高壓氣體所產生，然而，如果你連續2秒壓著不放，氣笛就會壞掉。在比賽之前，一群球迷聚在一起討論要如何安排氣笛聲。他們用一串僅含0與1的序列來表示，如果是1代表氣笛要響，如果是0代表氣笛不叫（此時呼喊KO-RE-A）。為了確保氣笛不會壞掉，序列中不可以有連續2個1存在。

現在你的任務是：給你一個正整數n，請你算出長度為n的序列中（僅含0、1）可以有多少種安排的方式不會出現連續2個1。例如：n=3，答案是5種（000,001,010,100,101可被接受，而011,110,111則否）。

Input

輸入的第一列有一個正整數代表以下有幾組測試資料。每組測試資料1列，含有1個正整數n（n < 51）。

Output

對每組測試資料輸出長度為n的序列中，不含連續2個1的序列有多少種，每組測試資料後請輸出一空白列。輸出格式請參考Sample Output。

　

Sample Input Sample Output 
3
3
1
50
 
Scenario #1:
5

Scenario #2:
2

Scenario #3:
32951280099
