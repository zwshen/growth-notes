Q10282: Babelfish

你剛剛搬到另一座城市，那裡的居民說一種外國語言。還好，你有字典可以幫助你翻譯。

Input

輸入首先是字典的內容，最多不會超過100000列。每列包含2個單字（Word，全部小寫且長度不會超過10個字元），第1個是英文，第2個是外國語言。接著有一空白列然後才是需要翻譯的外國語言的單字（每個一列）。在字典中，沒有任一個外國語言單字會出現超過1次。

請參考Sample Input

Output

請利用輸入字典資料，將待翻譯的外國語言單字翻成英語。如果在字典中找不到，請輸出"eh"

參考Sample Output

Sample Input

dog ogday
cat atcay
pig igpay
froot ootfray
loops oopslay

atcay
ittenkay
oopslay

Sample Output

cat
eh
loops
