Q375: Inscribed Circles and Isosceles Triangles

給你2個實數

  • B     等腰三角形的底長
  • H     等腰三角形的高
你的任務是計算以下的值到小數點後6位。
  • C     一連串的等腰三角形的內接圓圓週長的和。其中第一個內接圓與底及兩腰切，第二個內接圓與第一個內接圓及兩腰切，依此類推。為了要限制計算的範圍，半徑小於0.000001的內接圓均不再列入計算。

對於那些三角幾何有點生疏的人：三角形的內接圓為三個角的角平分線的交點，此點到三個邊的垂直距離均相同（也就是內接圓的半徑）。

Input

輸入的第一列有一個正整數代表以下有幾組測試資料。每組測試資料一列，含有2個實數（B H）。請參考Sample Input。

Output

每組測試資料輸出一列C的值（到小數點後6位），請注意：小數點的位置一定在第7行。測試資料間請空一列，請參考Sample Output。

Sample Input

2

0.263451 0.263451

0.987 0.54125

Sample Output

     0.827648

     1.700383
