Q10167: Birthday Cake

Lucy和Lily是雙胞胎，媽媽買了一個蛋糕因為今天是她們的生日。現在我們把蛋糕放在一平面座標中，蛋糕的中心位於 (0,0)，並且蛋糕的半徑是100。

在蛋糕上面有2N個草莓（1 <= N <= 50），媽媽想要用刀子把蛋糕切成同樣大小的兩塊（當然是通過中心，筆直的切一刀），為了公平起見，2塊蛋糕上面草莓的數目必須一樣。你能幫助她嗎？

注意：草莓的座標 (x,y)，x,y均為整數。你必須回答2個整數A,B，使得AX+BY=0 這一條直線代表媽媽所切的一刀。A,B的範圍必須在 -500~500之間。另外，草莓不可剛好位於這一條直線上。

Input

輸入包含多組測試資料。每組測試資料的第一列含有1個整數 N,接下來的2N列每列有2個整數，即代表蛋糕上2N個草莓的座標(x,y)

若 N=0代表輸入結束。請參考Sample Input。

Output

對每組測試資料輸出一列含有A及B，如題目所述。每組測試資料至少會有一組答案。

Sample Input Sample Output 
2
-20 20
-30 20
-10 -50
10 -5
0

0 1
