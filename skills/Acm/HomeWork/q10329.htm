Q10329: Combinatorial Expression

在組合數學中 C(n,r)代表在 n 件東西中取 r 件東西的方法數。數學定義如下：

              n!
C(n,r) = ----------------
           (n-r)! * n!

現在你的任務是寫一個程式去計算好幾個C(n,r)相乘的結果去除以好幾個C(n,r)相乘的結果。

Input

輸入含有多組測試資料。每組測試資料的第一列含有2個整數 N, M（N,M<=100），接下來的N列每列有2個整數 n , r，代表一個C(n,r)。再接下來的M列每列也是有2個整數 n, r（ 0 <= r <= n <= 5000）。你的任務就是將前N列的C(n,r)相乘，再去除以後M列C(n,r)相乘的結果。以Sample Input第一組測試資料為例說明：他代表 (C(10,5) * C(10,4) * C(10,3)) / (C(10,7) * C(10,6) * C(10,5))

Output

對每組測試資料輸出一列。如果無法整除請輸出 0。如果答案超出100位數請輸出 -1。否則請輸出答案。
