Q10233: Dermuba Triangle

下圖中的三角形皆為正三角形，邊長=1。每個三角形都有編號（從0開始）

這個問題是要請你求出編號n和編號m的三角形之間的距離（也就是2個三角形重心的距離）。

Input

輸入的每一行測試資料有2個整數n,m（ 0 <= n,m <= 2147483647）。

Output

輸出編號n和編號m的三角形之間的距離，請四捨五入到小數點後3位。

Sample input

0 7
2 8
9 10
10 11
100 200
2147483647 0

Sample Output

1.528
1.528
0.577
0.577
3.464
45226.010
