Q10013: Super long sums

我們現在要來做大數的相加，1000個字元長度的數字已經不算什麼了，你的任務是做2個大數（最多1000000個字元長度）的相加。

Input

第一列有1個整數N，代表接下來有多少組測試資料。

每組測試資料以一列包含整數M為開頭。（1 <= M <= 1000000）M代表要做相加的2個整數的長度。為了要使該2個數字長度相等，會在較短的數字前補0。接下來的M列，每列有兩個字元，以空白分隔。以Sample Input的第一組測試資料為例說明：M=4，要相加的兩個數分別是0463以及4287。

Output

每列測試資料輸出該2數相加的結果。測試資料間請空一行，請參考Sample Output。

Sample Input

2

4
0 4
4 2
6 8
3 7

3
3 0
7 9
2 8

Sample Output

4750

470
