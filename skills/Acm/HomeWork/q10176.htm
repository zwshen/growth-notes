Q10176: Ocean Deep! Make it shallow!!

你是一個偵探，在這個問題中你要面對的是一個叫做Rampell-Stilt-Skin人，比較困難的是他是一個死人。幾天前有人謀殺了他，而追查兇手的線索都記錄在他的一本日記中。這位先生是位難以理解的人，他在他的日記中記錄了一些事情，之後有一串二進位的數字（長度可能到達10000）。如果這個數字可以被 131071 這個質數整除的話，那麼這個記錄就是真的，否則，這個記錄就是假的。你的任務就是要判斷這個數字是否可被131071整除。

Input

輸入包含多組測試資料（二進位的字串）。

每組測試資料從一列的第一個位置開始，但是可能會有很多列。每組測試資料以#為結束。每列的長度不會超過100。請參考Sample Input。

Output

對每一組測試資料，如果可以整除，請輸出YES，否則請輸出NO。

Sample Input

0#
1010101#

Sample Output

YES
NO
